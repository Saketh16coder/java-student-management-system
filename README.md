# java-student-management-system


This is a simple console-based Java application demonstrating Core Java skills such as:
- Classes and Objects
- Arrays
- Methods
- Scanner-based user input
- Basic CRUD operations

## Features
- Add student details
- View all students
- Search student by ID
- Menu-driven program

## How to Run
1. Install JDK
2. Save the file as StudentManagement.java
3. Compile using:
   javac StudentManagement.java
4. Run using:
   java StudentManagement
